# Scripts e Botoes

Os arquivos `.bat` da raiz foram mantidos na raiz para ficarem faceis de encontrar.

## Uso Mais Comum

- `ABRIR_APP_LOCAL.bat`: testar mudancas.
- `GERAR_PACOTE_PRODUCAO.bat`: criar versao para publicar.
- `PUBLICAR_LOCCOBURGER.bat`: enviar site para Hostinger.

## Supabase

- `PUBLICAR_FUNCAO_SEFAZ_SUPABASE.bat`: publicar funcao SEFAZ.

## Manutencao

- `LIMPAR_TESTES_LOCCOBURGER.bat`: limpar dados de teste.

## Observacao

Os scripts `.ps1` sao os arquivos internos que fazem o trabalho.
Os `.bat` existem para voce dar dois cliques com mais facilidade.
